# Cotizador C&E

Aplicación simple hecha en Streamlit para cotizar obras según:
- Precio del m²
- Zona Norte (+10.000 por m²)
- Zona Sur (+20.000 cada 300 km)
- Distancia en kilómetros
- Adicional opcional
- Datos del cliente y del asesor

Incluye la opción de generar presupuesto descargable en PDF.
